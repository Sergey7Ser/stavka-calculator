# Калькулятор ставки эксперта

> В репозитории также живёт [`astralium.html`](astralium.html) — калькулятор сметы на натуральный камень и работы для компании **ASTRALIUM** (материал, производство, кромки, монтаж, укладка керамогранита, коэффициенты сложности; два уровня прайса, все цены редактируются, данные — в `localStorage`). Тоже один самодостаточный файл без зависимостей.

Помогает экспертам, мастерам и фрилансерам посчитать **честную стоимость своего часа** и превратить её в цену за услугу, задачу или день. Для тех, кто вечно занижает себя.

Оформлен в фирменных цветах **«Опоры Бизнеса»**.

## Что умеет

- **Мастер по шагам** — вопрос за вопросом (для тех, кто «не решается»): профессия → доход → часы → налог → % оплачиваемого времени → честная ставка.
- **Пресеты профессий** — разработчик, психолог, дизайнер, маркетолог, копирайтер, репетитор, фотограф + свой вариант. Подставляют типовые услуги и реалистичную базу.
- **Полный режим** — базовая ставка, прайс-лист услуг (порог / рекомендую / премиум), юнит-экономика.
- **Проверка загрузки времени** — план в часах против оплачиваемых часов: влезает или нет.
- **Обоснование цены клиенту** — по параметрам услуги собирает готовое сообщение для копирования.
- Данные хранятся только в браузере (`localStorage`), ничего не уходит на сервер.

## Технически

- Один самодостаточный файл `index.html` — весь CSS/JS/картинки внутри. **Без сборки, без зависимостей.**
- Запуск: открыть `index.html` в любом браузере. Или залить на любой статический хостинг.
