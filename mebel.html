Как пользоваться Мебельной сметой

Расчёт по изделиям → смета → коммерческое предложение в PDF. Прямо на замере, с телефона.

1. Быстрый старт на замере

  1. Вверху впиши клиента (имя, телефон) и адрес (город, улица, дом, квартира).
  2. Добавь изделие (Кухня, Шкаф-купе…) — строка «Изделие:» → + Изделие.
  3. Выбери вкладку и введи количество (п.м., м², штуки) — сумма считается сама.
  4. Внизу виден итог и срок изготовления.
  5. Жми Создать КП → и Скачать PDF — покажи или отправь клиенту прямо на месте.

2. Изделия

Один заказ — несколько изделий. Строка «Изделие:» → + Изделие («Кухня», «Шкаф-купе», «Гардеробная»). У каждого своя смета, а в КП — разбивка «сколько стоит каждое изделие».

3. Разделы и модульный режим

Вкладка 📏 Модульные · п.м. — быстрая прикидка «кухня за погонный метр» (эконом/стандарт/премиум) прямо при клиенте. Точный расчёт — по вкладкам Корпус · Фасады · Столешница · Фурнитура · Работы и монтаж. Цену в любой строке можно править на месте — запомнится.

4. Поиск

Строка 🔍 Поиск сверху: введи «фасад эмаль», «петля», «столешница кварц» → тап по подсказке → перескочит на нужную строку и поставит курсор в количество.

5. Свой прайс из Excel

Кнопка 📚 Справочник → ⬇ Скачать шаблон → впиши в Excel свои позиции и цены → ⬆ Загрузить свой (Excel). Файл читается прямо в телефоне, никуда не отправляется. Вернуть исходный — ↺ Стартовый.

6. Свои позиции

В конце каждой вкладки блок «➕ Добавить свою работу»: название → единица (п.м., м², шт, компл) → цена → количество → Занести в смету ✓.

7. Срок изготовления

Блок 🛠 Срок изготовления — впиши дни на изготовление и на доставку/монтаж. Общий срок соберётся сам и попадёт в КП.

8. Скидка и предоплата

Блок ％: скидка клиенту (отдельной строкой в смете) и предоплата (в условия оплаты КП).

9. Сохранение смет

💾 Сохранить смету — номер присвоится сам. 🗂 Мои сметы — весь архив с поиском по адресу/клиенту и пометкой «кому отправил». Клиент передумал — открыл старую, поправил, сохранил: создастся дубль, прежняя версия останется.

10. КП, PDF и CRM

Создать КП → — готовый документ с названием твоей компании (впиши один раз — запомнится). Скачать PDF → отправь клиенту. Выгрузка: ⬇ JSON (для CRM) — смета прикрепляется к карточке клиента; CSV — для Excel.

Данные хранятся только в этом браузере — на другом устройстве их не будет. Важные сметы дополнительно сохраняй в PDF.
