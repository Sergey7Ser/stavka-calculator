# Проект: калькуляторы-услуги (Опора Бизнеса / ASTRALIUM)

Фабрика одностраничных фирменных калькуляторов, которые превращаются в продукт:
расчёт → смета → коммерческое предложение (PDF) → публичная ссылка на поддомене.

## Карта репозитория

| Файл | Что это | Где живёт в проде |
|---|---|---|
| `index.html` | Калькулятор ставки эксперта («Опора Бизнеса»: синий #142634 + золото #D9BF85) | https://oporabusiness-sistema.ru/stavka.html (VPS владельца) |
| `astralium.html` | Калькулятор натурального камня ASTRALIUM: смета + КП с PDF | https://kamen.oporabusiness.ru (GitHub Pages) |
| `brand/` | Логотипы «Опоры Бизнеса» | — |
| `.github/workflows/pages.yml` | Автодеплой на GitHub Pages при пуше в `main` | — |

## Железные правила

- **Один самодостаточный HTML-файл** на калькулятор: весь CSS/JS внутри, без сборки,
  без CDN и внешних зависимостей. Данные пользователя — только `localStorage`.
- Владелец — не программист, общение на русском. Названия кнопок интерфейсов пиши
  **по-русски**, рядом в скобках точную английскую надпись: «Сохранить (Save)».
- Деплой: пуш в `main` → GitHub Pages обновляется сам (~2 мин). Поддомены — CNAME-записи
  на reg.ru → `sergey7ser.github.io`. `index.html` содержит переадресацию по hostname
  (kamen.* → astralium.html) — сохраняй её при правках.
- В этой облачной песочнице внешняя сеть закрыта (прокси отдаёт 403), SSH наружу тоже.
  Проверка живых URL — только глазами владельца или через GitHub MCP/API.

## Как проверять работу

Headless-скриншоты и e2e: глобальный playwright
`import { chromium } from 'file:///opt/node22/lib/node_modules/playwright/index.mjs'`,
браузер `/opt/pw-browsers/chromium-*/chrome-linux/chrome`. Не ставь playwright в проект
(появятся package.json/node_modules — репозиторий должен оставаться чистым).
